Cluster Configuration

3 control plane nodes, 9 workers. Cilium CNI with eBPF kube-proxy replacement. Talos Linux v1.8.

Networking

BGP peering with upstream routers. Direct server return load balancing. Wireguard encrypted mesh between sites.

Storage

TiKV distributed storage with JuiceFS mount. 2TB NVMe per node, erasure coded across 3 zones.
